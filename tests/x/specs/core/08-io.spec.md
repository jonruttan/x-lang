## write

### writes an integer

```scheme
(write 42)
```
---
    42

### writes a string with quotes

```scheme
(write "hello")
```
---
    "hello"

### writes a symbol

```scheme
(write (lit hello))
```
---
    hello

### writes a list

```scheme
(write (lit (1 2 3)))
```
---
    (1 2 3)

### writes a nested list

```scheme
(write (lit (1 (2 3))))
```
---
    (1 (2 3))

### returns nil

```scheme
(do (def r (write 42)) (newline) (null? r))
```
---
    #t

## read

### reads an integer

```scheme
(do (def x (read)) x) 42
```

### reads a symbol

```scheme
(do (def x (read)) x) hello
```

### reads a list

```scheme
(do (def x (read)) x) (1 2 3)
```

### reads a string

```scheme
(do (def x (read)) x) "world"
```

## read-char

### reads a single character

```scheme
(do (def c (read-char)) (char? c))
```

### returns nil on end of input

```scheme
(do (read-char) (null? (read-char)))
```

## current-line

### returns positive integer

```scheme
(> (current-line) 0)
```
---
    #t
